HTML: 

Wrap groups of content in separating elements, like <article> <section> or <div>

Using a class on repeated elements (like the images on the home page) can help with styling



CSS: 

Height and Width for images and elements in CSS can be controlled with px, %, and vw

Padding: between content and border, Margin: outside the border (between box and surrounding boxes)
  
margin: auto; will center content, if it doesn’t work try making sure the element is a block level element with display: block; and setting the width of the element

Flexbox is very helpful for centering content too 

Centering Guide: https://css-tricks.com/centering-css-complete-guide/

Flexbox Guide: https://css-tricks.com/snippets/css/a-guide-to-flexbox/

CSS selectors can help narrow down where you are applying styles

Class attributes in HTML can be repeated, helpful for styling repeated things. Use a period before the class name to reference in CSS

ID attributes in HTML can only be used once, helpful for styling specific things. Use a hashtag before the id name to reference in CSS
